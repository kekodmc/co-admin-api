co-admin-api
===============

配合项目[co-admin-vue](https://github.com/kekodmc/co-admin-vue)一起使用
===============

> 基于thinkphp 6.0+ <br/>
> 运行环境要求PHP7.1+ <br/>
> mysql 8.0 (低于8.0需要修改sql文件部分表编码) <br/>

## 安装

~~~
composer install
~~~

## 文档

[thinkphp 6 开发手册](https://www.kancloud.cn/manual/thinkphp6_0/content)
